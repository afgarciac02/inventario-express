## Roadmap

  ### run local server:
    run: npm run dev

  ### run server:
    start: npm start

  ### run ts to js compiler 
  ```bash
    build: npm tsc
  ```
    
    - the js project will stay in the build folder
    
 ## Installation

  ### Install my-project with npm
   **Node.js:**
    - https://nodejs.org/en/download/
   **Download dependencies:**
    ```bash
      cd my-project
      npm install
    ```
